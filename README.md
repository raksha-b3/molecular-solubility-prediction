## Dataset
The dataset used in this project is the ESOL (Delaney) solubility dataset.
Direct download (CSV):

Source: DeepChem repository

This repository contains a learning-based mini project created to explore machine learning applications in cheminformatics.
This project predicts aqueous solubility (logS) of chemical compounds using molecular descriptors
and a Random Forest regression model.

Tools used:
- Python
- RDKit
- Scikit-learn
- Pandas
- Matplotlib / Seaborn
